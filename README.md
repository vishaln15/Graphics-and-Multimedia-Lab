# Graphics and Multimedia Lab

## How to execute

- Navigate through every directory and execute the `cpp` file.

- Execute the following command in the terminal

```
g++ filename.cpp -o basic -lfreeglut -lopengl32 -lglew32 -lglu32 && filename
```
